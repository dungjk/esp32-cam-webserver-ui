# ESP32 CAM SERVER UI
---
ESP32 camera server UI

## Build
NPM favor
```sh
$ npm install
$ npm run build
```


YARN favor
```sh
$ yarn install
$ yarn build
```

## Development

NPM favor
```sh
$ npm run watch
```

YARN favor
```sh
$ yarn watch
```

## Debugging
During development we use sourcemap to debug ts/tsx code dirrectly on browser (Chrome, FireFox, Edge).
